# Game 核心概念详解

## 内功（Inner Game）

### 需求感（Neediness）
**定义**：你需要对方来validation你的自我价值。
**表现**：秒回消息、过度殷勤、对方冷淡时焦虑、过度投资。
**对策**：拥有自己的核心生活（事业、爱好、社交圈），对方是你生活的增量而非全部。

### 高地位心态（Abundance Mentality）
**定义**：你有很多选择，所以不急于在一个选项上all-in。
**效果**：行为自然松弛，不会因为对方一次冷淡就方寸大乱。
**培养**：扩大社交圈、和更多异性自然互动、提升自身价值。

### 专注自身（Self-Amusement）
**定义**：你说话做事首先是让自己开心，而不是取悦对方。
**效果**：看起来魅力十足——因为真正有趣的人都"自得其乐"。

## 外功（Outer Game）

### 推拉（Push-Pull）
- **拉**：肯定、赞美、靠近
- **推**：否定、调侃、拉开
- **节奏**：3:7（推3分，拉7分）到 5:5（对等）
- **禁忌**：拉太假（油腻），推太狠（伤人）

### DHV（Demonstration of Higher Value）
**不是炫耀**，而是自然地让价值被看到：
- **被动DHV**：别人提到你的好事情（"上次你们团建我听说你讲得特别好"）
- **叙事DHV**：讲一个有趣的故事，顺带流露你的社交圈/能力/见闻
- **行为DHV**：你对餐厅/酒吧/活动的选择本身就彰显品味
- **社交DHV**：你是那个调节气氛的人、大家认可的人

### Neg（轻微打压）
**目的**：发出信号"我对你并不完全买账"，打破对方的高社交地位。
**前提**：必须是轻微的、幽默的，不是攻击性的。
**示例**：
- 对方指甲很精致 → "你的美甲挺好看的，就是颜色有点像我外婆的窗帘😄"
- 对方聊了一堆自己的工作 → "你工作起来的样子和平时判若两人啊"

### 冷读（Cold Reading）
**定义**：基于观察给出"很了解她"的感觉。
**公式**：观察到一个细节 + 大胆推断 + 留出修正空间
**示例**：
- "你表面上很开朗，但我感觉你其实是个比较慢热的人。"
- "你选的这个包暴露了你的审美水平不错，你应该是个挺挑剔的人吧？"

## 情绪循环与框架

### 情绪摆动（Emotional Rollercoaster）
两个人的吸引力不是在"稳定"中建立的，而是在情绪高低起伏中建立的。
**好情绪**：开心、好奇、被认可、被关注
**坏情绪**：微妙的失落、困惑、被挑战、不服气
**理想状态**：你让她大部分时间开心，偶尔有一点点不确定感。

### 舒适度 vs 吸引力
```
吸引力太高 → 没有舒适度，像表演
舒适度太高 → 没有吸引力，像闺蜜
```
**平衡点**：70%舒适 + 30%张力

### 框架层博弈
- **朋友的框架**：友善、安全、没有性张力
- **玩家的框架**：轻浮、不确定、低承诺
- **男人的框架**：有态度、有原则、有保护欲
- **筛选者的框架**：你在考察她，不是她在选择你

**关键**：以筛选者框架为基底，根据场景切换。
